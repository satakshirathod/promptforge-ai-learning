# PromptForge – AI Prompt Engineering Learning Platform

PromptForge is a web-based learning platform designed to help users understand the basics of **Prompt Engineering** and how prompts influence AI-generated responses.

This project demonstrates how structured prompts improve the quality of AI outputs and provides educational resources for beginners.

---

## Project Overview

PromptForge provides an interactive learning environment where users can explore different prompt techniques and understand how to communicate effectively with AI systems.

The platform includes explanations, examples, and a simple playground where users can experiment with prompts.

---

## Features

- Dark themed modern UI
- Explanation of different **Prompt Engineering techniques**
- Comparison between **weak prompts and strong prompts**
- Overview of popular **AI tools**
- Interactive **Prompt Playground**
- Best practices for writing effective prompts

---

## Pages Included

- **Home** – Introduction to Prompt Engineering  
- **Prompt Types** – Explanation of Zero-shot, Few-shot, Chain-of-thought and Instruction prompts  
- **Prompt Examples** – Comparison of weak vs strong prompts  
- **AI Tools** – Information about ChatGPT, Claude, Gemini and Midjourney  
- **Playground** – A simple prompt testing interface  
- **Best Practices** – Guidelines for writing better prompts
