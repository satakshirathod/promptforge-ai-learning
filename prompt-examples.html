Prompt Examples

Prompt examples help users understand the difference between weak prompts and strong prompts. Well-structured prompts provide more context and instructions, helping AI models generate more accurate and useful responses.

Weak Prompt

Write about AI

This prompt is too short and lacks context. The AI model may generate very general or unclear responses because the task instructions are not specific.

Strong Prompt

Explain Artificial Intelligence in 5 bullet points for beginners.

This prompt clearly defines the topic, format and audience. As a result, the AI model can generate a more structured and informative response.

More Prompt Examples

Weak Prompt

Tell me about programming.

The prompt is vague and does not specify the programming language, topic, or level of explanation.

Strong Prompt

Explain the basics of Python programming with simple examples for beginners.

This prompt provides clear context, making it easier for the AI to generate a useful and structured response.
